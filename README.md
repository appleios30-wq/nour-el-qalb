# نور قلبك 🌙

تطبيق موبايل للأذكار المطلقة مع تأثيرات نيون متوهجة

## المميزات

### 1. الأذكار المطلقة
- 8 أذكار أساسية بألوان نيون مختلفة
- قابلة للإضافة لاحقاً
- فضائل كل ذكر

### 2. المسبحة الإلكترونية
- عداد رئيسي بألوان نيون
- تأثير شعاع ضوء عند كل ضغطة
- حلقة تقدم دائرية
- عند الاكتمال: انفجار احتفالي + دورة جديدة
- اهتزاز خفيف

### 3. الخلفيات الديناميكية
- **الليلي**: شجرة في وسط الصحراء + بحر + قمر + شخص جالس
- **الصباحي**: رصيف مزدحم + سحب + توهج الشمس البرتقالي
- **الغروب**: تصميم مميز يحفز على أذكار المساء

### 4. أسماء الله الحسنى
- 99 اسماً مع التعريفات والدلائل
- البحث والتصفية
- تأثيرات نيون مميزة

### 5. "سرك في بير"
- تدوينات مخفية برقم سري
- الرقم السري يُنشأ من عدد الضغطات على الشاشة
- خلفية ألوان مبهجة

### 6. الإعدادات الأولية
- اختيار الجنس (ذكر/أنثى)
- إدخال العمر
- اختيار المنطقة السكنية

## التقنيات

- **Flutter**: إطار العمل الرئيسي
- **Hive**: للحفظ المحلي
- **Google Fonts**: للخطوط العربية
- **Flutter Animate**: للتأثيرات الحركية
- **Confetti**: للتأثيرات الاحتفالية

## التثبيت

```bash
# استنساخ المشروع
git clone [repository_url]

# الانتقال لمجلد المشروع
cd nour_el_qalb

# تثبيت الاعتماديات
flutter pub get

# تشغيل التطبيق
flutter run
```
